# AI Claude Analyze

Claude Code 系统提示词工程的逆向分析。基于 Claude Code 源码，拆解大模型编程助手背后的设计模式。

## 内容

`Claude提示词工程解析/` 目录包含 16 篇分析文章：

| 编号 | 主题 | 核心发现 |

| 00 | 全景概览 | 七步分析框架，从"你是谁"到"你会做什么" |
| 01 | 分层组装与缓存优化 | 系统提示的静态/动态分区与 prompt cache |
| 02 | 身份塑造与安全围栏 | 三种身份前缀 + 安全围栏的门控机制 |
| 02a | 身份锚定-全部提示词 | 10 层身份矩阵的完整提示词逐条列出 |
| 03 | 行为引导与编码风格 | 工具使用的行为准则与错误处理流程 |
| 04 | 工具优先级与防呆设计 | 7 类工具的优先级排序与降级机制 |
| 05 | 子 Agent 编排 | Agent 递归派生与任务分配模式 |
