Claude 的本质是什么？是一个法师，施展咒语，指挥大模型。

## 电影原话收集

古一的核心台词：

1. **"A spell is nothing more than a formula. A tool to channel universal energies."**
   - 咒语不过是一个公式，一种引导宇宙能量的工具。

2. **"The language of the mystic arts is older than civilization."**
   - 神秘艺术的语言比文明更古老。

3. **"It's not about your strength. It's about precision."**
   - 这不是力量的问题，是精度的问题。
   - 对应：Prompt Engineering 不是堆 token，是精准控制。

4. **"Forget everything you think you know."**
   - 忘掉你以为你知道的一切。
   - 对应：打破"AI 就是聊天"的认知，提示词是工程。

5. **"We harness energy drawn from other dimensions of the multiverse."**
   - 我们驾驭从多元宇宙其他维度汲取的能量。
   - 对应：大模型的能力来自海量训练数据（另一个"维度"）。

## 内容方向

### 抖音视频结构设想

**开场**（3 秒钩子）：
> "古一法师说过一句话：咒语不过是一个公式，一种引导宇宙能量的工具。今天我告诉你，Claude 的提示词就是现代的咒语。"

**正文**（3-5 个技巧点，每个 15-20 秒）：

1. **分层组装 = 咒语的起承转合**
   - 不是一口气念完，而是分层、有节奏
   - 静态层 = 咒语的固定部分（可缓存）
   - 动态层 = 施法时根据敌人变化的部分

2. **工具优先级 = 法器的选择**
   - 古一有扇子、悬浮斗篷、阿戈摩托之眼
   - 不同法器用于不同场景，不能拿扇子去劈柴
   - Claude 有 36 个工具，每个都有适用场景

3. **子 Agent = 召唤术**
   - 至尊法师可以召唤帮手
   - 探索 Agent = 探子（只看不动）
   - 验证 Agent = 审计员（只管挑毛病）
   - 规划 Agent = 军师（只出主意不动手）

4. **上下文压缩 = 记忆魔法**
   - 长对话 = 长时间的战斗，记忆会模糊
   - 压缩 = 把回忆写成笔记，保留精华
   - 九段式摘要 = 法师的战斗日志

5. **安全围栏 = 卡玛泰姬的规矩**
   - 古一有规矩：不能用黑暗维度的能量
   - Claude 也有规矩：不能做破坏性操作
   - 咒语的力量需要约束，没有约束的力量是灾难

**结尾**：
> "所以学 Prompt 不是在学怎么跟 AI 聊天，是在学怎么当一个现代的至尊法师。你写的每一行指令，都是一个公式，都在引导宇宙能量。"

## 深层洞察

### 提示词工程的本质

- 不是"写话术"，是"写程序"
- 不是"调参数"，是"设计行为"
- 不是"给指令"，是"构建约束体系"

### 与传统编程的区别

| 传统编程 | 提示词工程 |
|---------|-----------|
| 编译器严格执行 | 模型"理解意图"后执行 |
| 语法错误 = 报错 | 模糊指令 = 行为不确定 |
| 确定性输出 | 概率性输出 |
| 函数调用 | 工具调用 |
| 变量作用域 | 上下文窗口 |

### Claude 提示词工程的 9 大技巧 → 9 种"咒语类型"

1. 分层咒语（缓存优化）
2. 身份咒语（定义自我）
3. 行为咒语（约束动作）
4. 法器咒语（工具选择）
5. 召唤咒语（子 Agent）
6. 变形咒语（输出风格）
7. 记忆咒语（上下文压缩）
8. 灵魂咒语（会话记忆）
9. 条件咒语（特性开关）
