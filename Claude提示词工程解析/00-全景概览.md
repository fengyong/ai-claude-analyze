## 架构总览：提示词的分层管道

Claude Code 的提示词系统不是"一段大 prompt"，而是一个**分层管道**：

```
┌─────────────────────────────────────────────────────┐
│                  System Prompt 构建                    │
│                                                       │
│  ┌──────────┐  ┌──────────┐  ┌──────────┐            │
│  │ 静态层    │  │ 动态层    │  │ 环境层    │            │
│  │(可缓存)   │  │(会话级)   │  │(实时)     │            │
│  └────┬─────┘  └────┬─────┘  └────┬─────┘            │
│       │              │              │                  │
│       └──────┬───────┴──────────────┘                  │
│              ▼                                         │
│     SYSTEM_PROMPT_DYNAMIC_BOUNDARY                    │
│     (静态/动态分界线，优化缓存命中率)                      │
│              │                                         │
│              ▼                                         │
│     ┌────────────────┐                                 │
│     │ Output Style   │  ← 可选覆盖层                    │
│     │ Overlay        │    (解释模式 / 学习模式)          │
│     └────────┬───────┘                                 │
│              │                                         │
└──────────────┼─────────────────────────────────────────┘
               ▼
┌──────────────────────────────────────────────────────┐
│                  Tool Prompt 注入                      │
│                                                       │
│  每个工具自带 prompt.ts → 工具可用时注入                   │
│  ~36 个工具 prompt + 6 个内置 Agent prompt              │
└──────────────┬─────────────────────────────────────────┘
               ▼
┌──────────────────────────────────────────────────────┐
│              上下文管理 & 持久化层                       │
│                                                       │
│  ┌──────────────┐  ┌──────────────┐                   │
│  │ Compact      │  │ Session      │                   │
│  │ (上下文压缩)  │  │ Memory       │                   │
│  │ 9 段式摘要    │  │ (跨轮次记忆)  │                   │
│  └──────────────┘  └──────────────┘                   │
└──────────────────────────────────────────────────────┘
```

## 关键数据

- **系统 prompt 总长度**: ~915 行核心 + 环境信息 + 工具 prompt
- **工具 prompt 数量**: 36 个（每个工具一个 `prompt.ts`）
- **内置 Agent 数量**: 6 个（Explore, Verification, GeneralPurpose, Plan, ClaudeCodeGuide, StatuslineSetup）
- **Output Style**: 2 种内置覆盖（Explanatory, Learning）+ 插件/用户自定义
- **缓存分界**: `SYSTEM_PROMPT_DYNAMIC_BOUNDARY` 将 prompt 分为可缓存的全局部分和会话级动态部分
- **Ant vs External**: 至少 7 处 prompt 内容对 Anthropic 内部用户 vs 外部用户有差异
