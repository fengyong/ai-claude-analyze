## 整体洞察

这些新角度揭示了 Claude Code 不仅在"提示词"层面做工程，而是在 6 个层面同时发力：

```
提示词层 → Hooks 外置层 → 权限/分类器层 → 沙箱物理层 → 实验平台层 → 可观测性层
```

- 提示词层（00-15）：用语言引导行为
- Hooks 外置层（16）：用事件驱动替代 token 消耗
- 权限/分类器层（17/18/29）：用 ML 和协议语义替代模型猜测
- 沙箱物理层（19）：用 OS 级隔离替代提示词约束
- 实验平台层（20）：用 A/B 测试替代经验调优
- 可观测性层（23）：用数据反馈替代主观评估
